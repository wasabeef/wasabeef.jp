## Usage

```sh
# install & update
npm install -g firebase-tools
```

## firebase init
```sh
# login
firebase login:ci
# setup
firebase init
```

## Flutter web 
```sh
$ flutter build web --release
$ flutter run -d chrome
```

## firebase
```sh
# test
firebase serve --only hosting
# deploy
firebase deploy
```
