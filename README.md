## Usage

```sh
# install & update
npm install -g firebase-tools
```

## firebase init
```sh
# login
firebase login:ci
# setup
firebase init
```

## Flutter web 
```sh
flutter build web --web-renderer html --release
flutter run -d chrome --web-renderer html
```

## firebase
```sh
# test
firebase serve --only hosting
# deploy
firebase deploy
```
